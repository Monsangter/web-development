자바는 1995년 선 마이크로 시스템에 의해 출시된 프로그래밍언어이며 컴퓨팅 플랫 폼이다.
현대 대부분의 자파 어플들이 자바 실행환경과 어플을 합쳤지만 아직도 많은 웹사이트들이 자바 설치를 요구한다. 이 웹사이트는 데스크탑 어플로서 자바를 계속 필요로하는 사람들을 위해 만들 어 졌다. 자바를 배우려는 개발자는 dev.java, 사업주는 oracle.com/java에 들어가 봐라.

자바는 다운로드 무료인가?
자바는 개인적 용도로도 무료이며 개발용도로도 무료이다.

왜 최신 자바패치가 나온 분기에 바로 업그레이드 해야하는가.

성능과 안정성, 보안성을 올린다.

자바닷컴에서 자바 소프트웨어를 다운로드 하면 나는 뭘 얻는가

자바실행환경을 얻는다 jre version 8 이게 자바 닷컴에서 자바를 다운로드했을때 얻을 수 있는 거다.

자바 실행환경은 jvm 자바 가상머신으로 구성 돼 있다. 자바 플랫폼의 핵심 클래스 이며, 자바 플랫폼 라이브러리를 재공한다. jre 는 자바 소프트웨어의 일부이며 자바 웹스타트 어플을 지원하는 웹 브라우저로부터 필수적으로 다운로드 받게한다. 개발 툴은 jre의 일부가 아닌 jdk의 일부이다.

자바 플러그인 소프트웨어란 무엇인가?
자바 플러그인 소프트웨어는 jre런타임 실행환경은 자바언어로 적힌 몇몇 어플들을 몇몇 브라우저를 거쳐 실행 될 수 있게 한다. 자바 플러그인 소프트웨어는 독립적 프로그램이 아니며 따로 인스톨 될 수도 없다. 

자바 가상머신이라는 용어를 들었는데 이것도 자바의 일부인가?
자바 가상머신은 자바소프트웨어의 한부분일 뿐이다. 이것은 어플 실행에 관여한다. 자바 가상머신은 자바소프트웨어 다운로드에 의해 구현되며 jre의 일부로서 자바어플리케이션 실행을 돕는다.
